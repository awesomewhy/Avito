## Как скачать проект

1. Склонировать репозиторий
```sh
git clone https://github.com/awesomewhy/Avito.git
```
# Запуск в докере

Нужно перейти в папку где находится проект

1. Собираем проект 
```sh
mvn package
```
2. Билдим докер файл
```sh
docker compose -d --build
```
3. запускаем
```sh
docker compose up
```
Возможно у кого-то так 
```sh
docker-compose up
```
